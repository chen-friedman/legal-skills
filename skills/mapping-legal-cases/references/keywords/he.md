## 12. דפוסי פרטיות — Privacy-Sensitive Patterns (Israeli-specific)

Always flag and mask in PRIVACY_FLAGS.md:

| סוג | דפוס |
|---|---|
| תעודת זהות / ת"ז / ת״ז | `\b\d{9}\b` near "תעודת זהות" / "ת.ז." / "ת"ז" |
| דרכון / מספר דרכון | 6-10 alphanumerics near "דרכון" |
| חשבון בנק ישראלי | `\b\d{2,3}[\s-]?\d{3}[\s-]?\d{6,}\b` near "בנק" / "חשבון" |
| IBAN ישראלי | `\bIL\d{2}[\s-]?\d{4}[\s-]?\d{4}[\s-]?\d{4}[\s-]?\d{4}[\s-]?\d{3}\b` |
| מספר טלפון ישראלי | `\b0\d{1,2}[\s-]?\d{7}\b` |
| מספר טלפון בינלאומי | `\+972[\s-]?\d{1,2}[\s-]?\d{7}` |
| מספר תיק רפואי | digits near "מספר תיק רפואי" / "מ.ת.ר." |
| אבחנה רפואית | proximity to "אבחנה" / "ICD" codes |

**Masking in Hebrew output:** mask values the same way — first digits + asterisks (e.g. `123******`).
