# Example Output

Illustrative samples of what `mapping-legal-cases` produces when run on the sample case folder at `examples/sample-case-folder/` in the repository root.

These files are **abbreviated** for readability — real output is typically more detailed.

Files in this folder:
- `DOCUMENTS.sample.md`
- `TIMELINE.sample.md`
- `CLAIMS.sample.md`
- `DEADLINES.sample.md`

See `references/output-templates.md` for the complete set of 10 document templates and their full structure.
